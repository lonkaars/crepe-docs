# Code style

- Indent using tabs
- Wrap long lines at column 80
- Diacritical marks should be represented in ASCII using LaTeX syntax instead
  of using UTF-8 (i.e. `fa\c{c}ade` instead of `fa√ßade`)
- Images should be placed in the [img/](./img/) folder
